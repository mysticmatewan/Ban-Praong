2. Phpmyadmin Inserting Data For Staff

  1. Open phpMyAdmin: Start by launching the Apache and Mysql server. (such as XAMPP).
  2. Select the Database: Launch http://10.1.14.53/phpmyadmin/, select the database where you want to insert the staff data. You can find the list of databases on the left-hand side of the interface.
  3. Choose the Table: Click on the table that will store the information for each year.
  4. Navigate to the Insert Tab: Once inside the table, click on the "Insert" tab at the top of the page. This tab allows you to add new records to the table.
  5. Enter Data: You will see a form with fields corresponding to the columns of your table. Fill in each field with the appropriate data. For example, you might enter:
    • NO. PENDAFTARAN (Registration Number)
    • NAMA BAHAN / ARTIFAK (Name of Material / Artifact)
    • JENIS PEROLEHAN (Type of Acquisition)
    • TARIKH PEROLEHAN (Date of Acquisition)
    • SUMBER (Source)
    • DESKRIPSI SUMBER (Source Description)
    • KLASIFIKASI UTAMA (Main Classification)
    • SUB-KLASIFIKASI (Sub-Classification)
    • KUANTITI (Quantity)
    • DESKRIPSI (Description)
    • NAMA PEREKOD (Recorder's Name)
    • GAMBAR BAHAN (Material Image)

  6. Submit the Form: After entering all the necessary data, scroll down and click the "Go" button. This action will insert the data into the table.
  7. Verify the Insertion: To ensure that the data has been inserted correctly, click on the "Browse" tab to view all the records in the table, including the new entry.


3. Tweaking ENUM and Creating Table MS ACCESS

Microsoft Access is a powerful database management system (DBMS) that allows users to create, manage, and manipulate databases through a user-friendly interface. While MS Access does not natively support the ENUM data type, commonly found in databases like MySQL, you can achieve similar functionality through other methods such as using lookup tables or custom validation rules. When designing a database in MS Access, understanding how to effectively create and manipulate tables is crucial. Tables are the backbone of any database, storing the actual data that users interact with. Creating tables involves defining the structure of the data, including specifying data types, field sizes, and relationships between tables. In this guide, we will explore how to simulate ENUM-like behavior in MS Access by using lookup tables or validation rules. We will also cover the basic steps involved in creating tables in MS Access, ensuring that your data is organized efficiently and accurately. This will allow you to build robust databases tailored to the needs of your application. Whether you're new to MS Access or looking to refine your skills, this introduction will provide a solid foundation for managing data in Access.
